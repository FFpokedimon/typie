# Typie
Тайпи - клавиатурный тренажер для детей, которые еще только приучаются к клавиатуре и не могут свободно печатать. Формат будет представлять собой различные игры, которые требуют взаимодействия с клавиатурой и развивают навыки печатания.

## Техническое задание:
- Сверстать шаблон base.html, включающий в себя шапку сайта и бэкграунд [+]
- По подобию base.html сверстать главную страницу index.html, таблицу лидеров leaderboard.html, профиль profile.html, страницу друзей friends.html [~]
- Сделать окно регистрации
- Сверстать песочницу sandbox.html и прописать ее логику game-logic.js [+]
- По подобию песочницы сделать уровни levels.html [-]
- Подогнать все страницы под шаблон base.html при помощи jinja [~]
- Спроектировать и построить архитектуру (главный файл main.py, папка для БД db, таблицы БД models, статический контент static, шаблоны templates) [+]
- Инициализировать и подключить БД typie.db [+]
- Создать таблицы БД users и games [+]
- Написать простейшую логику сайта main.py (адресация и загрузка статического контента) [+]
- Создать системы аккаунтов и друзей [-]
- Настроить запись и изменение данных в БД [-]
